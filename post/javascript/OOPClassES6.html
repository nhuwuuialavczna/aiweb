Cách sử dụng class trong ES6

    I. Giới thiệu
        - Từ trước đến nay có lẽ các bạn đã quen với cú pháp JS thuần rồi. Tuy nhiên việc khởi tạo và thiết lập kế thừa khác rườm rà và khó khăn cho các bạn
        không hiểu rõ. Do đó ở phiên bản ES6. JS cung cấp thêm cho chúng ta từ khóa class và các từ khóa liên quan để thao tác với đối tượng tốt hơn
        và gần giống với các ngôn ngữ lập trình khác
    II. Thao tác với đối tượng trong ES6
        1. Chúng ta khởi tạo class Person như sau:
            class Person{
                constructor(name,age){
                this.name = name;
                this.age = age;
            }
                sayHello(){
                    return "xin chao toi là: "+this.name;
                }
            }

            Hihi, khá đơn giản đúng không. Trong đó từ khóa constructor để khởi tạo 1 constructor cho class person. và sayHello là 1 phương thức trong đối tượng Person

        2. Tiến hành sử dụng tính năng kế thừa như sau:
            - Tạo thêm đối tượng Child extends Person:
                class Child extends Person{
                    constructor(name,age,hobby){
                        super(name,age);
                        this.hobby = hobby;
                    }
                    sayHello(){
                        return "Xin chao e là: "+this.name+" e thích: "+this.hobby;
                    }
                }
                Trong đó có từ khóa super để kế thừa các thuộc tính của Person và chúng ta đang override lại phương thức sayHello
        3. Nếu đoạn code trên được viết với JS thuần thì như thế nào:
