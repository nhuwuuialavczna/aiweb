Vì trang web là web tĩnh để bỏ qua mọi gián đoạn khi xem bài viết. Tuy nhiên đổi lại sẽ bị mất rất nhiều nguồn tài nguyên khác đó là webservice
    - Hiện tại trang web đang có 3 webservice:
        + http://serviceaicode.azurewebsites.net ( NodeJS)
        + http://javaide.azurewebsites.net (Java) (đang xây dựng)
        + http://tools.azurewebsites.net (C#) (đang xây dựng)
    - Chú ý: Không thể truy cập các trang web trên của có thể get data qua API thôi.
    - CHỉ có admin mới thấy được nội dung này
